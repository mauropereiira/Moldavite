Moldavite

Your thoughts, forged from cosmic impact.

Download for Mac Read the guide

macOS 10.15+ · Apple Silicon & Intel · Free & open source

Moldavite is a rare green gemstone formed 15 million years ago when a meteorite struck Earth. The impact fused terrestrial and extraterrestrial matter into something entirely new.

This app carries that spirit. Your notes live only on your Mac—fused with your workflow, shaped by your thinking, belonging entirely to you.

No cloud. No accounts. No tracking.

Everything a thinking tool needs

A fast, native-feeling editor with the connective tissue of a knowledge base — and none of the lock-in.

Local-first by design

Notes are plain Markdown files in your Forge at ~/Documents/Moldavite/. Sync it, back it up, or open it in any other tool.

Rich editing

Markdown shortcuts, headings, task lists, images, highlights, code blocks, and a slash menu for quick actions.

Daily & weekly notes

One click to today's note, created automatically and cleaned up if left empty. Weekly notes straight from the calendar.

Connected thinking

[[Wiki links]] with a backlinks panel and a graph view. #tags with global rename. Folders when you need structure.

Find anything fast

Full-text search with ranked snippets, plus a command palette / quick switcher on ⌘P for notes, actions, and plugin commands.

Apple Calendar, read-only

Today's events beside your notes via native EventKit — permission-gated by macOS, never written to, never uploaded.

Templates

Custom templates with {{date}}, {{time}}, and {{day_of_week}} variables, pinnable for one-click starts.

Multiple Forges

Keep separate vaults — Work, Personal, anything — and switch between them instantly. Each Forge has its own plugins and pinned notes.

Plugins

Add your own commands to the palette and slash menu with a small, permissioned JavaScript API. Build one →

Private by architecture, not by promise

Moldavite doesn't have a privacy policy so much as a privacy design: there is no server, no account, and zero telemetry. Nothing to leak, because nothing leaves.

Plain files you own

Every note is a Markdown file on your disk with owner-only file permissions. Delete the app and your notes are still just files.

Zero telemetry

No analytics, no crash reporters, no phone-home. The only network request the app makes is checking GitHub for signed updates.

Encrypted note locking

Lock sensitive notes behind a password — encrypted with AES-GCM and Argon2 key derivation, with brute-force rate limiting and auto-lock.

Encrypted backups

Export your entire vault as a password-protected encrypted archive. Restore it anywhere, on your terms.

Signed auto-updates

Updates are code-signed and cryptographically verified before install, so what you run is exactly what was released.

Plugin consent, pinned

Plugins run only after you approve their permissions — and consent is pinned to a SHA-256 hash of the plugin's code, so any change re-prompts you.
